ID

Los ID se utilizaban para declarar qué parte de la página se estaba presentando separando cada sección con DIV, sin embargo ahora es más común usar las etiquetas semánticas.

Aún se utilizan en algunas etiquetas que van a ser únicas para especificar las características de esas etiquetas en lo individual

ID sólo se declara en una etiqueta y no se puede repetir en otra parte de la página web.

Para llamar al ID en CSS, se antepone el signo de número (#) al nombre del ID

#nom_ID

Medidas absolutas

Son medidas que no cambian si la pantalla cambia de tamaño, es decir, no se adapta por lo que siempre se verá del tamaño inicial; estas principalmente son los PIXELS

Medidas relativas

Se adaptan a las pantallas porque están en función de la misma. Pueden ser con PORCENTAJE, REM o EM

Jerarquía de los atributos

Las características que se le definen a una etiqueta pueden quedar empalmadas varias entre lo generar, por ejemplo a un div o en lo específico como con un ID

Entre más específico sea, tendrá más prioridad, por ello, las características definidas directamente en línea son la cabeza de la lista.

Clases

Se pueden definir a varias etiquetas que van a compartir formato.

Para llamar a la clase en el archivo de CSS se antepone el punto (.) al nombre de la clase

.nom_clase

Se pueden utilizar en diferentes tipos de etiquetas que compartan formato.
