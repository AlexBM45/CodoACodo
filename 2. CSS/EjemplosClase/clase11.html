Siempre la página tiene una fuete por defecto, pero se pueden cambiar. CSS contiene varios conjuntos de fuentes que se van enlistando jerárquicamente para indicar cual será el principal por utilizarse y cuales son opciones alternativas en caso de que el navegador no lea el anterior a este.

Lo recomendable es tener un máximo de tres fuentes en toda la página para que se vea uniforme el diseño.

Google ofrece varios tipos de fuentes para que sean utilizados por todos los desarrolladores, sólo se copia una parte del código que se debe pegar en el HTML para llamarlo.

Es la forma más común actualmente para darle fuentes a la página.
