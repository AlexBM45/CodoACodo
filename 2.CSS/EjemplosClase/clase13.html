Media Queries

Medidas estándar con las que se trabaja para que el diseño se adapte a todo tipo de pantallas.

Se trabaja a partir del dispositivo más pequeño al más grande.

Los media queries se escriben al final de CSS siempre que todos esten en el mismo archivo, aunque se pueden dividir en diferentes archivos.

Para los diseños responsivos se utilizan medidas relativas que se adaptan a la pantalla que los contiene y así es más fácil.
