Flexbox

Es una metodología que incorpora CSS para trabajar con las posiciones. Toma lo mejor de las posiciones, los display y todas las funciones para las posiciones y los resume en unos pocos comandos.

Se trabaja con FILAS y COLUMNAS

Se debe tener un contenedor padre que va a contener a sus hijos y estos son los que se van a mover, ya que es al padre al que se le aplica DISPLAY:FLEX; y así poder manipular a los hijos.

FLEX tiene diferentes atributos, el que viene por defecto es WRAP e indica como se acomodan las cajas en la página en caso de que no alcance el espacio de forma horizontal.

NOWRAP modifica los tamaños para que todos los elementos se sitúen en la pimera fila del contenedor padre.

También existen las opciones FLEX-START, FELX-CENTER, FLEX-END para indicar como se van a localizar los elementos en las filas, si al final, al inicio o en el centro.

ALING

Manipula los elementos hijos en forma vertical y les indica en que posiciones podrían estar

ALING-SELF permite modificar esta posición sólo de un hijo.
