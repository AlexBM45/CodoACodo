Imágenes

Todos los archivos gráficos poseen dimensiones de ancho y alto, que se pueden manejar en HTML mediante height y width

Imágenes con enlaces

Se pueden ligar links a imágenes mostradas en el página, pero aveces esto crea un borde del color del link, lo cual pueden no ser muy estético

Esto se puede eliminar con el parámetro border el cual le indica a la etiqueta cuantos pixeles ocupará el borde

Para indicar los espacios libres entre la imagen y el resto de la página se utilizan vspace y hspace
