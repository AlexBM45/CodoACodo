Bootstrap

Es un Framework para frontend. Se pueden bajar los archivos o linkearlos desde la web.

Son una base para empezar un proyecto web permitiendo flexibilidad en el diseño. Ya tiene preparado:

  1. Grid
  2. Estilos para las fuentes
  3. Componentes visuales pre-armados
  4. Responsive design
  5. Formatea automáticamente la página

Permite ahorrar tiempo al momento de crear la página.
