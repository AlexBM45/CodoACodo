JavaScript

Lenguaje de programación que interpreta el navegador, el motor V8. Débilmente tipado, lo que significa que no es necesario declarar el tipo de dato que se usará. La versión fuertemente tipada de JS es el Typescript.

El punto y coma en este caso es opcional y o afecta el funcionamiento del código si se escribe o no al final de cada instrucción.

Se puede trabajar desde la consola o desde le DOM.

El código de JS se puede insertar en el HTML con la etiqueta SCRIPT o se puede escribir en un archivo independiente y linkearlo a la página en HTML, como en el caso de CSS:

script src="nombre-archivo.js"

Esta línea normalmente va al final del archivo, justo antes de cerrar la etiqueta BODY para se aplique a todo la página y no corte el despliegue del HTML y del CSS con alguna instrucción de JS que así lo indique y de resultados no deseados.

La extensión de los archivos son .js

Los comentarios en JS se pueden hacer con doble barra (//) para una sola línea y con barra asterisco en apertura y cierre (/* ... */) para comentarios de varias líneas.

Operaciones

Los Strings se pueden concatenar con otro string o con un número con el símbolo de suma (+)

En el caso de presentarse dos números, estos se pueden sumar, restar y todas las operaciones matemáticas que existen.
